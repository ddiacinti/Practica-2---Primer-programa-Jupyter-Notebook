## Instalación
1. Crear el entorno virtual:
python -m venv .venv

2. Activar el entorno virtual:
.venv\Scripts\activate

3. Instalar las dependencias:
pip install jupyter[lab]

## Ejecución
Para abrir Jupyter Lab:
jupyter lab
